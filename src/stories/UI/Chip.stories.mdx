Chip are used for things like tagging where a 'chip' of information is presented with an optional remove button.

## Basic example

<Story
  name="a) Chips"
  parameters={{
    docs: {
      source: {
        code: dedent`
<div className={chip.container}>

  <Chip onRemove={() => console.log("removed")}>
    Something else chipper
  </Chip>
</div>
);`,
      },
    },
  }}

</Story>
